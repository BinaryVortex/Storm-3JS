# Storm-3JS

A dynamic storm simulation built using Three.js and GLSL shaders. This web-based project creates a visually stunning, interactive storm effect with clouds, lighting, and atmospheric movement.

## 🌟 Features

- **Real-time Rendering**: Smooth, animated storm clouds using fragment shaders
- **Interactive Controls**: Mouse movement affects the storm dynamics
- **Scroll Interaction**: Vertical scrolling influences the storm's appearance
- **Video Capture**: Press 'D' key to record and save the animation as a WebM video
- **Responsive Design**: Adapts to different screen sizes and resolutions

## 🛠 Technologies Used

- **Three.js**: 3D graphics library for WebGL rendering
- **GLSL Shaders**: Custom vertex and fragment shaders for storm effects
- **Texture Mapping**: Noise and cloud textures for realistic rendering
- **CCapture.js**: Library for capturing WebGL animations

## 📁 Project Structure

- `index.html` - Main HTML file with embedded shaders
- `script.js` - Three.js setup and animation logic
- `style.css` - Basic styling for the full-screen experience

## 🎮 Controls

- **Mouse Movement**: Interact with the storm by moving your cursor
- **Scroll**: Use your mouse wheel or scroll to affect the storm's behavior
- **Video Capture**: Press the 'D' key to start/stop recording the animation

## 🏃‍♂️ Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/BinaryVortex/Storm-3JS.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Storm-3JS
   ```

3. Open `index.html` in your web browser.

Note: The project loads textures from external URLs, so it requires an internet connection for full functionality.
